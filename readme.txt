Lets you add tabs to your post and pages using a shortcode. 

== Description ==

To add tabs just have to insert this shortcode on your page / post:

[mtabs]

[mtitle]Title 1[/mtitle]

Content 1

[mtmore]

[mtitle]Title 2[/mtitle]

Content 2

[mtmore]

[mtitle]Title 3[/mtitle]

Content 3

[/mtabs]

You can also add type = "2" in the shortcode and displayed with a different style (accordion)

[mtabs type="2"]

[mtitle]Title 1[/mtitle]

Content 1

[mtmore]

[mtitle]Title 2[/mtitle]

Content 2

[mtmore]

[mtitle]Title 3[/mtitle]

Content 3

[/mtabs]

You can also create a shortcode with the new option on the tinymce called "MT Tabs".

== Installation ==
See http://codex.wordpress.org/Managing_Plugins#Installing_Plugins

== Upgrade Notice ==
If you've made ​​changes to the CSS of the plugin, please use a style.css file in your theme folder, in a subfolder called mt-tabs. 

This will tell the plugin to load the CSS from your theme and will not be overwritten when you upgrade.

== Frequently Asked Questions ==

= How to disable the plugin CSS? =

Creating a new folder inside your theme with the name "mt-tabs" and style.css file you can customize the CSS to 100% of the plugin.

= How to translate the plugin into my language? =

You can edit the "mt-tabs-en_EN.po" file to find the plugin folder lang and translate it to your language poedit easily.
